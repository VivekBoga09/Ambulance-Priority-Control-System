# Smart Ambulance Green Corridor System 🚑

A full-stack web application designed to create **smart green corridors for ambulances** by prioritizing emergency vehicles at traffic intersections. The system helps reduce ambulance travel time by dynamically controlling traffic signals during emergencies.

# Project Overview

Traffic congestion can delay ambulances and impact emergency medical response. This project provides a **smart traffic management solution** that detects emergency vehicles and automatically prioritizes their route by managing traffic signals in real time.

# Features

*  Emergency ambulance route prioritization
*  Dynamic traffic signal control system
*  Real-time route monitoring
*  Instant updates for traffic signal status
*  Interactive web interface for monitoring

# System Workflow

1. Ambulance emergency request is generated in the system.
2. Backend processes the request and identifies the route.
3. Traffic signals along the route are automatically switched to **green**.
4. The system updates the traffic status in real time through the web interface.
